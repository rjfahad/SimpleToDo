## How to Contribute

We welcome contributions from the community! Here's how you can help:

- **Code**: Submit pull requests for bug fixes or new features
- **Translations**: Help translate the app into more languages
- **Documentation**: Improve README, FAQs, or other docs
- **Bug Reports**: Report issues on GitHub
- **Feature Requests**: Suggest new features via GitHub Issues
- **Testing**: Test new releases and provide feedback

See the main [README](README.md) for detailed contribution guidelines.
